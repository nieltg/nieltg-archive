## Compilation

To compile:

```
$ make
```

This command will generate floppy image on `bin/floppy.img`.

## Testing

Bochs can be used for testing.

```
$ bochs -f opsys
```

Then, press `c` to continue.

You can use another virtual machine other than Bochs.
